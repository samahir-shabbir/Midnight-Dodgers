## 🎮 About the Game

**Midnight Dodgers** is a simple 2D arcade-style game developed in **C++** using **SFML**. The player controls a car and avoids enemy vehicles while scoring points. The game includes multiple screens such as a welcome screen, instructions screen, gameplay screen, pause menu, safe mode, and game over screen.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🚗 Player-controlled car | Smooth keyboard-based movement |
| 👾 Enemy vehicle spawning | Randomized obstacles entering the road |
| 💥 Collision detection | Real-time hit detection between player & enemies |
| 🏆 Score tracking | Live score counter as you survive longer |
| ❤️ Lives system | Limited lives before game over |
| ⏸️ Pause & resume | Pause menu during gameplay |
| 🛡️ Safe mode | Temporary invincibility after a collision |
| 🖥️ Multiple game states | Welcome, instructions, gameplay, pause, and game over screens |
