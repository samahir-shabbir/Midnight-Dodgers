# Midnight-Dodgers - Source Code

This branch contains the complete source code and assets for the Midnight-Dodgers game, developed in C++ using SFML.

## Requirements

- Visual Studio 2022 or later
- SFML Version 2.6.2
- Windows Operating System

## Project Structure

- `CarDodgingGame.cpp` - Main game source code
- `CarDodgingGame.vcxproj` - Visual Studio project file
- `dodge Game.slnx` - Visual Studio solution file
- `.png` files - Game sprites and assets
- `PressStart2P-Regular.ttf` - Game font

## Setup Instructions

1. Install SFML.
2. Open `dodge Game.slnx` in Visual Studio.
3. Configure SFML include and library paths if required.
4. Build the project.
5. Run the application.

## Gameplay

- Control the player's car using the keyboard.
- Avoid incoming enemy vehicles.
- Earn points by surviving longer.
- Use pause and resume functionality during gameplay.
- Try to achieve the highest score possible.

## Features

- Player-controlled car
- Enemy vehicle spawning
- Collision detection
- Score tracking
- Lives system
- Pause and resume functionality
- Safe mode after collisions
- Multiple game states

## Assets

All required images, sprites, and font files are included in this branch.
