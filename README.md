# Midnight-Dodgers
A simple 2D arcade-style game developed in C++ using SFML. The player controls a car and avoids enemy vehicles while scoring points. The game includes multiple screens such as a welcome screen, instructions screen, gameplay screen, pause menu, safe mode, and game over screen.
<br>
## Features
<br>
- Player-controlled car<br>
- Enemy vehicle spawning<br>
- Collision detection<br>
- Score tracking<br>
- Lives system<br>
- Pause and resume functionality<br>
- Safe mode after collisions<br>
- Multiple game states<br>
